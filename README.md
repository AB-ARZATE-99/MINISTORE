# 🛒 Mini Store Project

Este es un proyecto de práctica enfocado en maquetación web, utilizando tecnologías fundamentales para crear interfaces modernas y responsivas.

## 🚀 Tecnologías implementadas
- **HTML5**: Estructura semántica del contenido.
- **CSS3 (Grid/Flexbox)**: Implementación de sistemas de diseño para organizar elementos.
- **Git/GitHub**: Control de versiones y despliegue del proyecto.

## 🛠️ Características clave
- **Diseño en cuadrícula**: Uso de *CSS Grid* para crear una galería de productos adaptable.
- **Arquitectura de componentes**: Cada producto está encapsulado en una "tarjeta" reutilizable.
- **Responsive Design**: Ajuste visual para diferentes resoluciones.

## 📝 Aprendizajes destacados
- Implementación de `display: grid` con `grid-template-columns`.
- Gestión de assets externos mediante URLs de recursos.
- Resolución de conflictos de rutas entre HTML y hojas de estilo (CSS).
